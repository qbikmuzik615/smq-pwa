# SMQ PWA

Touch-reactive WebGL fluid splash + quiz game. Installable as PWA from Safari on iOS/iPadOS.

## Deploy (GitHub Pages)
1. Push this repo to GitHub
2. Settings → Pages → Deploy from branch → main / root
3. Open the Pages URL in Safari on iPhone or iPad
4. Share → Add to Home Screen

## Fluid Controls
- **Start blank** — black screen until first touch
- **Visible time** — how long fluid trails persist
- **Color rotation** — hue cycling speed
- **Bloom glow / Glow intensity** — luminous bloom effect
- **Splat radius** — size of each touch splat
- **Vorticity** — swirl expansion
- **Swirl power / Rainbow speed** — motion and color feel

Settings are persisted in localStorage across launches.
